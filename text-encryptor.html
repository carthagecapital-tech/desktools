Encrypt and decrypt text with AES-256-GCM. Nothing leaves your browser.

Encrypted output
AES-256-GCM · PBKDF2 key derivation
How it works

Your passphrase is turned into a 256-bit key using PBKDF2 with 100,000 iterations. Text is then encrypted with AES-256-GCM which provides both confidentiality and integrity. The output includes the salt and IV needed for decryption. All processing happens in your browser using the Web Crypto API.
